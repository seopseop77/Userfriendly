`uv sync` installs them by default.

## Decisions

- `[dependency-groups]` preferred over `[project.optional-dependencies]` for dev deps in uv
  workspaces — uv installs all dependency groups by default, extras require `--extra`.
- `alembic/` stays inside `packages/llm_tracker/` (1:1 with core schema); no change to path
  references needed since `%(here)s` in `alembic.ini` resolves to its own directory.
